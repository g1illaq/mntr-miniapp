# Mentor Community — дизайн-система и презентации

Презентации делаем как HTML-слайды в формате бренда: **квадрат 1:1
(1080×1080), тёмная тема, синий акцент**. Это даёт полный контроль над брендом,
ревью дизайна через скилл `design-review` и экспорт в PDF/PNG.

## Структура

```
brand/
  brand.md          # бренд: палитра, шрифты, компоненты, правила
  assets/           # logo.*, avatar.*, исходные PDF (кладёшь сюда)
design-system/
  tokens.css        # ЕДИНСТВЕННЫЙ источник правды: цвета, шрифты, размеры
  slides.css        # слайд 1:1 + компоненты (pill, нум-пилюли, метки, рамки, кольцо)
presentations/
  template.html     # эталон из 6 слайдов, демонстрирует все компоненты
README.md
```

## Как сделать новую презентацию

1. Скопируй `presentations/template.html` → `presentations/<тема>.html`.
2. Меняй контент внутри `.slide`. Стили и бренд берутся из дизайн-системы.
3. Готовые «кирпичи» (см. `brand.md` и шаблон):
   - моно-метка `.label`, заголовок `.display` / `<h1>`, текст `<p>` / `.lead`
   - `.pill--glass / --blue / --outline / --white`, `.steps`/`.step`, `.tags`/`.tag`
   - картинка `.figure`, выноска `.callout`, фон `.ring-bg` / `.glow` / `.grid-bg`
4. Меняешь бренд (цвет/шрифт) — правишь только `design-system/tokens.css`,
   применяется ко всем презентациям сразу.

## Ревью дизайна (скилл design-review)

В Claude Code:

```
Review the design of ./presentations/template.html
```

или `/design-review`. Скилл откроет файл в браузере (Playwright), снимет
скриншоты и выдаст аудит: типографика, контраст, отступы, иерархия — с правками.

## Просмотр в браузере

Открой HTML — `design-system/present.js` сам включит режим презентации:
один слайд вписан в окно, листание **← →**, кликом по половине экрана, свайпом,
пробелом; снизу счётчик и точки-навигация. Кнопка **⊞ Сетка** (или клавиша `g`)
переключает в обычную вертикальную ленту — она же нужна для экспорта.

> Лента сразу при открытии: добавь `#grid` в URL.

## Рендер / экспорт

- **PNG по слайдам** (как делаю при проверке) — Playwright-скриншот каждого
  элемента `.slide`. Удобно для постинга карусели.
- **PDF**: открыть HTML в браузере → Печать → «Сохранить как PDF», размер
  1080×1080, поля 0. В CSS есть `@media print` (один слайд = одна страница).

## Шрифты

Подобраны бесплатные близкие к брендовым: **Onest** (заголовки/текст) +
**JetBrains Mono** (метки). Подключаются с Google Fonts в `<head>` шаблона.
Будут реальные шрифты — заменим в `tokens.css` и подключим файлы.
