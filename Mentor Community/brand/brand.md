# Бренд Mentor Community

Заметки по бренду. Финальные значения цветов/шрифтов/размеров живут в
[`../design-system/tokens.css`](../design-system/tokens.css) — это единственный
источник правды. `brand.md` описывает контекст и правила.

## Суть

Знак — **кольцо с тремя соединёнными узлами** = сеть/сообщество менторов.
Вордмарк — `mntr / comm` строчными. **Логотип целиком белый** (знак + текст),
токен `--mc-logo`. Синий — только акцент в контенте, не в логотипе. Тон: тёмный,
технологичный, «нейро/клинично, но премиально». Один акцентный цвет — синий.

## Формат

- **Квадрат 1:1, 1080×1080** — карусели для соцсетей/Telegram (основной формат).
- 16:9 при необходимости добавим позже (общие токены, второй размер кадра).

## Типографика

> Подобраны **бесплатные** шрифты, близкие к брендовым (по твоему выбору).
> Если пришлёшь реальные — заменим на точные.

- Заголовки/текст: **Onest** (округлый гротеск), `--mc-font-heading` / `--mc-font-body`
- Метки КАПСОМ: **JetBrains Mono**, `--mc-font-mono` — used for eyebrow/футер

## Компоненты (в `slides.css`)

- **Моно-метки** (`.label`) — КАПС, разрядка, синие/приглушённые
- **Pill-карточки** (`.pill` + `--glass / --blue / --outline / --white / --ghost`)
- **Нумерованные пилюли** (`.steps` / `.step` / `.step__num`) — кружок с цифрой + текст
- **Теги** (`.tags` / `.tag`) — маленькие синие пилюли
- **Рамка картинки** (`.figure` + `--16x9` + `--ph` плейсхолдер)
- **Выноска** (`.callout`) — синяя плашка с иконкой
- **Фон**: мотив кольца (`.ring-bg`), свечение (`.glow`), сетка (`.grid-bg`)

## Ассеты

Складывай в `brand/assets/`:
- `logo.*` — логотип `mntr comm` (SVG/PNG с прозрачностью)
- `avatar.*` — аватарка (кольцо + 3 узла)
- исходные презентации (PDF) — для точной выборки цветов/шрифтов

Пока знак нарисован SVG-символом прямо в шаблоне (`#mc-mark`) — заменим на
настоящий файл логотипа, когда положишь.

## Правила

- Контраст текст/фон ≥ 4.5:1 (WCAG AA): off-white на near-black — ок.
- Отступы кратны 8 (8/16/24/32/48/64/80/96).
- Один акцент — синий. Красный только как семантика «плохо/предупреждение».
- Метки — всегда моно КАПСОМ; заголовки — Onest жирный.
